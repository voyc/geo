Model - Application Framework

Javascript, PHP, Postgres, AJAX

Features

  • User Authentication and Management, via the account submodule.
  • Model-View-Controller architecture.
  • Object-oriented design.
  • Event-driven process flow, using the Observer pattern in the jslib submodule.
  • Responsive design. Automatically adjusts to user's window-size.
  • A framework for AJAX services, using the Comm object in jslib submodule.
  • AJAX navigation honoring the browser's forward and back buttons, via the BrowserHistory object in the jslib submodule.
  • Popup dialogs, modeless and modal, provided by the minimal submodule.
  • Open Source.
  • Efficient, compact code.
  • Supports google closure compiler.

About
